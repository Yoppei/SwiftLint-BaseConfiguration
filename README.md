# SwiftLint-BaseConfiguration

## Usage

Add the following line into your `.swiftlint.yml`.

```yml
parent_config: https://raw.githubusercontent.com/Yoppei/SwiftLint-BaseConfiguration/v1.0.0/swiftlint-base.yml
```
